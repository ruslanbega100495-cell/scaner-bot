После запуска `START-ALL.bat`:

1. **Откроется окно** "Freelance Monitor" с логами
2. **Откроются 3 вкладки** в браузере:
   - Health check: http://localhost:3000/health
   - Скрапинг: http://localhost:3000/api/v1/jobs/scan?source=kwork
   - Заказы: http://localhost:3000/api/v1/jobs

## ⚠️ ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### "Docker не запущен"

**Решение:**
1. Запустите Docker Desktop
2. Или используйте WSL: `docker-compose up -d`

### "Порт 3000 занят"

**Решение:**
1. Закройте другие приложения на порту 3000
2. Или измените PORT в `.env`

### "Telegram не отправляет"

**Решение:**
1. Проверьте `.env` (токен и Chat ID)
2. Нажмите Start в @Agent_freelance_bot
